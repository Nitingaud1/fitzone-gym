# 💪 FitZone Gym Website

A modern and responsive gym landing page built using **HTML, CSS, and Bootstrap 5**.  
This project is designed to showcase a fitness/gym website with clean UI and responsive layout.

---

## 🚀 Features

- Responsive Navigation Bar
- Hero Section with background image
- About Us Section
- Services Section
- Membership Pricing Plans
- Trainers Section
- Contact Form
- Smooth responsive design using Bootstrap 5
- Hover effects on cards

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5

1. Clone this repository:

